# ⚛️ AtomCord

**Голосовой штаб нового поколения**

AtomCord — это современный аналог Discord с акцентом на голосовое общение и минималистичный дизайн.

## 🚀 Быстрый старт

```bash
# Клонирование
git clone https://github.com/YOUR_USERNAME/atomcord.git
cd atomcord

# Установка
npm run install:all

# Запуск в режиме разработки
npm run dev
